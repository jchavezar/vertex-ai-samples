🔑 Install user OAuth token

The Cloud Run service account has no Jira refresh token bound to the GE Atlassian connector. Without a user OAuth token, the Rovo/custom-MCP/ federated pipelines (B / C / CG / DG) will return auth errors.

On your laptop, run:
gcloud auth print-access-token --account=admin@jesusarguelles.altostrat.com …then paste the output below. Tokens last ≈60 min.
