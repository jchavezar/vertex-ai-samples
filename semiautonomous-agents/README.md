| [`discovery_engine_test`](./discovery_engine_test/) | Vertex AI Discovery Engine integration tests. |
| [`hierarchical_rag_nexus`](./hierarchical_rag_nexus/) | Hierarchical RAG pipeline using ADK agents. |
| [`light_ge_mcp_cloud_portal`](./light_ge_mcp_cloud_portal/) | Lightweight Grounding Engine + MCP cloud portal. |
| [`light_mcp_cloud_portal`](./light_mcp_cloud_portal/) | Lightweight MCP-connected cloud portal. |
| [`pgvector_document_nexus`](./pgvector_document_nexus/) | Document embedding and retrieval using pgvector. |
