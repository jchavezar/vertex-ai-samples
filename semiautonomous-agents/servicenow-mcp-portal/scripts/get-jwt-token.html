4. Test Commands

Copy these commands to test your services:

Test MCP Server

# Replace YOUR_TOKEN with the ID token above
curl -X POST http://localhost:8080/mcp \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"method": "tools/call", "params": {"name": "list_incidents", "arguments": {"limit": 5}}}'

Test Agent Engine

# Create session with token
curl -X POST "https://us-central1-aiplatform.googleapis.com/v1beta1/projects/YOUR_PROJECT/locations/us-central1/reasoningEngines/YOUR_AGENT_ID/sessions" \
  -H "Authorization: Bearer $(gcloud auth print-access-token)" \
  -H "Content-Type: application/json" \
  -d '{"userId": "test", "agentSession": {"state": {"USER_TOKEN": "YOUR_JWT_TOKEN"}}}'
