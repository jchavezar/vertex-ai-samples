# 🔍 Observability Nexus: The "Brain" of Antigravity Monitoring

Observability Nexus is a real-time monitoring and telemetry visualization hub for the entire Antigravity project suite. It acts as a central aggregator for events, API calls, and logs generated by distributed agents and microservices.

## 🏗️ Role in the Platform
While `nexus_telemetry` acts as the "sensor" (capturing data), **Observability Nexus** is the "Display/Brain." It provides:
1.  **Ingestion Server**: A high-performance FastAPI endpoint (`/ingest`) that receives telemetry asynchronously.
2.  **Real-Time Broadcast**: Uses WebSockets (`/ws`) to push every incoming event to the browser instantly.
3.  **Visual Dashboard**: A React-based UI (Port 5179) that shows a "Heartbeat" of everything happening in the platform.
4.  **AI Analysis**: Integrates with Gemini (Vertex AI) to allow users to "chat with their logs," identifying issues and summarizing system status via the built-in Chat Overlay.

---

## 🚀 How to Run

### **1. Launch Backend (Port 8145)**
```bash
cd backend
uv run uvicorn main:app --port 8145
```

### **2. Launch Frontend (Port 5179)**
```bash
cd frontend
npm run dev
```

---

## 🩺 System Integration
Typically, you don't call this application's code directly. Instead:
1.  Other apps import the **`nexus_telemetry`** package.
2.  They send their data to `http://localhost:8145/ingest`.
3.  This app captures and renders that data for you.

> [!TIP]
> Use the **AI Chat** feature in the dashboard to ask questions like:
> *"Why did my SharePoint agent fail 2 minutes ago?"* or *"Summarize the last 10 API calls from the Stock Terminal."*
